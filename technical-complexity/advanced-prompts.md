# 高级提示词模板

## Self-Consistency 方法

```
请解决以下问题：{问题}

请从不同角度思考并给出3个不同的解答路径，然后选择最合适的答案。

解答路径1：
[思考过程]

解答路径2：
[思考过程]

解答路径3：
[思考过程]

最终答案：
```

## Tree-of-Thoughts 方法

```
问题：{问题}

让我们构建一个思维树来解决这个问题：

根节点（问题分析）：
{问题分析}

分支1：方法A
- 子节点1：步骤1
- 子节点2：步骤2
- 子节点3：步骤3
结果：{结果A}

分支2：方法B
- 子节点1：步骤1
- 子节点2：步骤2
- 子节点3：步骤3
结果：{结果B}

评估并选择最佳方案：
```
